What is JSX, and why is it used?
Answer: JSX stands for JavaScript XML
It's a React syntax that lets us write HTML-like UI inside JavaScript, making components easier to read and build.


What is the difference between State and Props?
Answer: Props are read-only data that is passed from a parent to a component. State is internal data a component owns.


What is the useState hook, and how does it work?
Answer: useState is a React Hook that lets a function component store and update local state by returning the current value

How can you share state between components in React?
Answer: Share state by lifting it to the nearest common parent and passing it down via props.


How is event handling done in React?
Answer: Event handling in React is done by passing handler functions to JSX event props.
